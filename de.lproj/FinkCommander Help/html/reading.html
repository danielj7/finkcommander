Verstehen der Paketliste

Die Spalten Installiert, Binär, Unstable und Stable

Diese Spalten zeigen die Versionsnummern der Pakete an. Durch Vergleichen der Nummer in der Spalte Binär mit den Nummern desselben Paketes in den anderen Spalten kann festgestellt werden, ob mit dem Kommando Binär:Install die neueste Version oder zumindest die neueste Version aus Stable des Paketes in binärer Form installiert werden kann. Eine ausführlichere Beschreibung der Unterschiede zwischen den Kommandos im Binär und Source Menü finden Sie in der Hilfedatei Kommandos ausführen:Source vs. Binär.

Die dargestellten Versionsnummern sind die für die Installation zur Verfügung stehenden Nummern. Dabei ist berücksichtigt, ob Sie das Installieren von Unstable Paketen eingeschaltet haben.

Version

Diese Spalte ist im wesentlichen eine kompaktere Darstellung der oben genannten Spalten. Sie zeigt Ihnen immer die neueste erhältliche Version, sei dies in binärer Form oder als Source-Datei, eines Paketes.

Zustand

"Aktuell" bedeutet, dass Sie die neueste Version des Paketes installiert haben. "Veraltet" heisst, dass das Paket installiert, aber eine neuere Version erhältlich ist. "Archiviert" bedeutet, dass Sie dieses Paket zuvor installiert hatten und es nun als .deb Datei auf Ihrem System archiviert ist, oder dass Sie ein Paket erstellt, aber nocht nicht installiert haben (in Version 0.3.3a oder Šlteren Versionen von FinkCommander erschien diese Information in der spalte Installiert).

Pakete finden

FinkCommander stellt Ihnen zwei nützliche Werkzeuge zur Verfügung, um nach Paketen zu suchen, die Sie vielleicht installieren oder entfernen wollen:

  1. Sie können durch Anklicken der Spaltenköpfe die Paketliste nach den jeweiligen Kriterien sortieren, so wie Sie das von anderen Programmen gewohnt sind. Erneutes Klicken desselben Spaltenkopfes kehrt die Reihenfolge um.

  2. In der Grundkonfiguration enthält die Werkzeugleiste einen Filter, welchen Sie auf die meisten Felder der Liste anwenden können. Der Filter arbeitet stufenweise, das heisst mit jedem Buchstaben, den Sie eintippen oder löschen, wird die Liste so angepasst, dass nur Pakete angezeigt werden, welche Ihr Suchkriterium enthalten. Die Anzeige der Paketanzahl im Fenstertitel ändert sich ebenfalls und widerspiegelt die Anzahl der gefundenen Pakete.

Durch Kombination von Sortierung und Filterung können Sie also effektiv die Tabelle nach zwei Kriterien durchsuchen.

Übrigens: Sie können die Breite der Spalten stufenlos verstellen und sogar deren Reihenfolge vertauschen. Änderungen werden beim Beenden des Programms natürlich gespeichert.
