Configuring FinkCommander and Fink

The FinkCommander Preferences Panel is divided into six tab views: Commander, Fink, Downloads, Path, Display and Environment. The Fink and Downloads tab views allow you to change the fink configuration settings stored in the fink-basepath/etc/fink.conf file.

This help file describes the purpose of a preference item when it may not be self-evident.

Fink

Use unstable packages/unstable cryptography packages

You can use these items to switch easily and quickly between using unstable packages and not using them. After you make a change here and click OK or Apply, the table will update itself to reflect the change.

Keep the build directory

From the fink man page: The default is false, fink usually removes the source build directory after successfully building a package. If you want to keep the build directory, set this to true. Usually only developers have any need for this field.

Keep the root directory

From the fink man page: Similar to the KeepBuildDir option, this allows developers to keep the root directory around for debugging purposes.

Downloads

Use HTTP proxy/FTP proxy

Set these items to the appropriate values if you are running Fink from behind a firewall. What are the appropriate values? Heck if I know.

Passive mode FTP

Because active FTP is deprecated, and some servers require passive mode, it is recommended that you leave this option on.

Download method

Mac OS X comes with curl, so it's the default. You'll need to install wget or axel before switching.

Paths

Path to Fink files

Only change the settings here if FinkCommander was unable to locate the path to your Fink installation on startup.

Display

Limit output scrollback to ____ lines

Use this setting to keep FinkCommander from consuming excessive amounts of memory when performing long installs (e.g. gnome, kde).

Scroll to the last selection after sorting the table

If this option is selected, the table will automatically scroll the last selection back to its previous position after you sort the table contents, just like Mail.app.

Environment

Use this table to set environment variables needed for fink or apt-get to run on your system. A few default settings are established for you. If you lose functionality after making changes here, click Restore to bring back the default settings.
