Learning About Your Fink Packages

FinkCommander provides several ways to learn about the software packages you have installed with Fink. All of the commands are applied to the packages you have selected in the main window table.

File:Open .info/.patch

Also accessible by double clicking or dragging and dropping the selected packages.

Opens the files used by Fink to install packages from source. Double-clicking and drag and drop open both the .info and .patch files. Drag and drop can also be used to copy these files to Finder windows or the Desktop.

Tools:Browse Package Files

Also accessible from: contextual menu for table, toolbar button.

Opens a Finder-like window showing the files installed by Fink for the selected package. You can open multiple windows with this command, but it can only be applied to one table selection at a time.

Buttons in the upper left-hand corner allow you to switch between an outline or browser view. You can open files displayed in the window with File:Open Selected File(s) or by double clicking. You can also open files or copy them to Finder via drag and drop.

Using the browser view for the larger packages can consume a pretty big chunk of memory. The memory is released back to the system, however, when you close the window. The outline view, in contrast, has a much smaller memory footprint, which is why it's set as the default.

Tools:Go to Package Documentation

Also accessible from: contextual menu.

Opens a Finder window for the package documentation directory installed as part of the Fink package.

Tools:Show Package Info

Also accessible from: contextual menu, toolbar button.

Opens a window showing the Fink description for whichever package or packages are selected in the table. The description includes live links to the web page for the package and the package maintainer's email address.
