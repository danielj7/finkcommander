Known Bugs and Limitations

Interaction sheet display failure

In some cases the interaction sheet may not display when Fink is waiting for a response. This sometimes occurs because Fink has not automatically responded to a prompt, even though Preferences:Commander:Always choose default response to prompts is selected. More detail.If you exprience this problem, select Tools:Interact with Fink to enter a response. An optional Interact with Fink item is also available for the Toolbar.

Apt-get update failure

If you receive an error message after running apt-get update, try running Source:Scanpackages and then trying apt-get update again.

Tcsh package problem

The selection of tcsh is disabled, because FinkCommander stalls when trying to install this package. Installing from the command-line works fine. Please file a bug report if you encounter the same problem with another package.
